BeipySimple极简主题包
========
Rainmeter软件是轻巧的Win10桌面美化软件，BeipySimple极简主义桌面风格。

内部插件包含：

    Clock系统时间
    power开关机功能
    power buttons快捷图标方式
    Recycle bin极简文字回收站
    Search 百度快捷搜索框
    Settings暂时未设置此功能，
    Sidebar 吸附式隐藏快捷菜单
    SysTem 系统状态
    Visualizer 音频跳动装饰。
    Weather（未完成）
    默认主题排版辅助线1080（此功能仅支持16:9屏幕）
    排版辅助线 Help 2560
    （包含个人所收藏的几张壁纸）
